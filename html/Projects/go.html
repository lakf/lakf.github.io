Go

Go is a travel planning website - without the planning.

Description

Go is a web site which helps users translate their unspecified wants and desires into planned out travel trips. For example, a user may have emotional requirements such as wanting to feel one with nature, perhaps be in the sun, or be around other travellers - however they might find it difficult to recognise these desires when planning their own trip - and then even more diificult to actually satisfy these desires through a planned trip. Trip users select images - in a Pinterest style - which represent their intrinsic travel desires. It then allows certain fileters such as budget, number of days etc before using API integrations and AI to plan the perfect itinerary.

Tech Stack

This project uses a vanilla react website with a python flask backend, postgresql database and GPT-4.
