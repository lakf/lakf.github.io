World Cup Wordle

World Cup Wordle was an independent project completed in late 2022.

Description

The idea for this project was to create a copy of the popular Wordle game, however instead of guessing the letters of words the user would now be guessing the letters of the last names of footballers playing in the world cup 2022. In a "guess-who" manner, if the user guessed a player with a same attribute as the mystery player, this attribute would be revealed. For example, if a user guessed Rice (Declan Rice), but the mystery player was Kane (Harry Kane), then they would be shown the attributes that they share (English nationality, playing in English leagues).

This project was a great introduction in getting hands on with running a full stack architecture. I used a simple html, js & css frontend. The backend was made using python flask, and the database was a postgresql database. This database contained the one table which contained all of the players and their various attributes.

I deployed this project using Heroku to host the backend and database, with Vercel hosting the frontend.
