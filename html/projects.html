Projects

A brief section to give some taste into my CS projects thus far!

Rob Bank

Rob Bank is a single player game in which the user must input combinations of actions to unlock a vault and “rob the bank”. There are six sets of combinations which the user must pass through, if they fail to do so within the time limit they lose. Key hardware used includes the BBC Microbit and various extra components. All software was written in Make Code for Microbit and Processing.

This project had two aims. Firstly, it needed to be a fun and interactive game because a boring game is never played and thus redundant. Secondly, it needed to use a range of inputs and outputs to demonstrate the skills I had learned in both hardware and software. Therefore, it was key to think about how I could include many inputs and outputs but still use them in a meaningful way which contributes to, rather than clutters, the gameplay.

Shape Drawer

Shape Drawer is a program in which the user can draw predefined shapes onto the screen, choosing the colour through a colour setting menu, and choosing the size by clicking the mouse at an origin and releasing at the desired scale. The program has undo and redo functionality, where the user can undo an unlimited number of shapes and redo them as expected.

The objective of the Shape Drawer project was to firstly, develop my java object orientated design skills, but also to gain exposure to working with Java Swing. This was my first foray into building a GUI and I did so using a model - view - controller model.
